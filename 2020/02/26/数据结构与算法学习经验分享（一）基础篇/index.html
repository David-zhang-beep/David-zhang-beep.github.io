基本数据结构

1.STL stack(栈)

相关函数：

函数 功能
s.empty() 判断栈是否为空
s.top() 返回栈顶
s.push(x) 将x添加至栈
s.pop() 弹出栈顶
s.size() 栈存储元素的个数

2.STL queue(队列)

相关函数：

函数 功能
Q.empty() 判断队列是否为空
Q.front() 返回队首元素
Q.back() 返回队尾元素
Q.pop() 弹出队首元素
Q.push(x) 将x添加至队列
Q.size() 队列存储元素的个数

3.二叉堆

功能：维护一定数量的最大值或最小值（求中位数）

实现方法：

std::priority queue<int> big_heap //默认最大堆

std::priority queue<int, std::vector, std::greater> small_heap //最小堆``

4.STL set(集合)

功能：主要用于查重

实现方法：

std::set<int> test_set; //查询复杂度O(nlogn)

5.STL map(哈希)

功能：主要用于建立映射关系

实现方法：

std::map<ListNode*,int> node_map; //建立链表节点与整型数值的映射

6.STL sort(排序)

功能：主要用于排序处理

实现方法：

bool cmp(ListNode *a, ListNode *b){

return a->val < b->val;

}

std::sort(vecor.begin(), vector.end(),cmp); //将数组元素从小到大排序

7.总结

以上内容均为学习数据结构与算法过程中最常用到的数据结构。由于目前学习内容有限，只总结了部分内容，可供总结和查看。具体代码的练习还是得亲自做题巩固。
